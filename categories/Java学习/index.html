API 的概述

API(Application Programming Interface) ，应用程序编程接口 。JavaAPI 是一本程序员的字典，是 JDK 中提供给我们使用的 类的说明文档 。这些类将底层的代码实现封装了起来，我们不需要关心这些类是如何实现的，只需要学习这些类如何使用即可。所以我们可以通过查询 API 的方式，来学习 Java 提供的类，并得知如何使用它们。
